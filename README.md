
# UIC Final Year Project: BrokenHeart
Type: 2D Roguelike Game 

Group members: CKX(@CandiceCKX), WSH, XSH

## Intro

### 🕹️Control

- Press W,A,S,D to move the character.

- Press F to pick up items.

- Press 1,2,3,4 to use items.

- Press "Tab" to Switch Weapons.

- Click the left key on the mouse to attack.
